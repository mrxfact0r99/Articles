Xshotter: Capturing the Visuals

Xshotter GitHub Repository

Xshotter is designed to simplify the process of capturing screenshots from a list of target URLs. Whether you're looking to assess the visual elements of a web application or need to document findings for your bug bounty reports, Xshotter has got you covered. It's a command-line tool that's both easy to use and highly customizable.

With Xshotter, you can:

  • Visit a list of target URLs.
  • Take screenshots of web pages.
  • Save the screenshots for further analysis or documentation.

Check out the Xshotter README for detailed usage instructions and customization options.

Xtractor: Uncovering Hidden Clues

Xtractor GitHub Repository

Xtractor is another indispensable tool in your bug bounty arsenal. It's designed to extract HTML comments from web pages, which can often contain valuable information about a site's architecture, developer notes, or even security misconfigurations.

With Xtractor, you can:

  • Visit target URLs and parse HTML content.
  • Extract HTML comments for analysis.
  • Gain insights into the inner workings of web applications.

For a deep dive into using Xtractor effectively, head over to the Xtractor README.
